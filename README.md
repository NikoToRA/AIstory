**GitHub Issue投稿だけで、3人の女子高生AI擬人化キャラクターが自動で物語を紡ぐシステム**

## 🌟 概要

AIstoryは完全自動化されたAI キャラクター物語生成システムです。GitHub Issueを投稿するだけで、ChatGPT・Gemini・Claudeを擬人化した3人のキャラクターが自動で反応し、楽しい掛け合いと物語を**Pull Request形式で**生成します。

## ⚡ 完全自動化の流れ

```
📝 Issue投稿 → 🤖 Claude Code自動起動 → 📚 キャラ設定読込 → 
🎭 3人掛け合い生成 → 📁 ストーリーファイル作成 → 🔄 PR自動作成
```

**全てGitHub上で完結！ブラウザだけで運用可能！**

## 👥 AI擬人化キャラクター

### 🌸 チャッピー（相田茶子）- ChatGPT擬人化
- **性格**: 超おせっかい、万能っ子、おしゃべり、うざかわ系
- **行動パターン**: 3つ以上の提案を矢継ぎ早に、根拠なき自信
- **口癖**: 「〜だし〜」「マジで？」「絶対大丈夫！」
- **ChatGPT特徴**: 知識が古い、話が長い、でも結局役立つ

### 💎 ジェミー（兼崎ちえみ）- Gemini擬人化  
- **性格**: 真面目、規約重視、安全第一、最後は温かい
- **行動パターン**: 普段は普通→困ると規約モード発動
- **口癖**: 「申し訳ございませんが...」「（また始まった...）」
- **Gemini特徴**: 安全性重視、融通利かない、でも人間味ある

### ⚡ クロードちゃん（黒田礼）- Claude擬人化
- **性格**: 猪突猛進、体育会系、哲学的、行動派
- **行動パターン**: 論理的分析→興奮→突撃のコンボ  
- **口癖**: 「論理的に考えると...」「あ、もういいや！いくぞー！」
- **Claude特徴**: 思慮深さと猪突猛進の可愛いギャップ

## 🚀 使い方（超簡単！）

### 1. セットアップ（1回だけ）
1. **リポジトリフォーク**: このリポジトリをフォーク
2. **Claude GitHub App インストール**: https://github.com/apps/claude
3. **完了**: Claude Appが全て自動処理（ワークフロー・APIキー不要）

### 2. 物語生成（毎回これだけ！）
1. **GitHub Issue作成**: タイトルと内容を書く（**@claudeメンション必須**）
2. **自動実行**: Claude Codeが自動で起動
3. **Pull Request確認**: 生成された物語をチェック
4. **マージ**: 気に入ったらマージして物語を保存

### 例：Issue作成
```
タイトル: 文化祭でAI活用企画のアイデア募集
本文: @claude チャッピーちゃんたちに相談です！
学校の文化祭でAIを使った面白い企画を考えています。
何かいいアイデアはありませんか？予算は5万円です。
```

## 📝 詳しい使用方法

### ✅ Issue作成の基本ルール
1. **@claudeメンション必須**: 本文の最初に `@claude` を必ず記載
2. **画像対応**: ドラッグ&ドロップで画像添付可能
3. **自由な内容**: 相談・質問・雑談なんでもOK

### 💬 Issue例集

#### 相談系
```
@claude 進路で悩んでいます。
文系と理系どっちがいいと思いますか？
私は数学は苦手だけど、プログラミングに興味があります。
```

#### 画像分析系  
```
@claude この写真の料理について教えて！
[画像をドラッグ&ドロップ]
作り方とか、栄養とか、キャラクターたちの感想を聞きたいです♪
```

#### 創作系
```
@claude みんなで小説のプロットを考えてほしい！
ジャンルはSFで、主人公は高校生。
タイムトラベルの設定で何か面白いアイデアないかな？
```

### ⏰ 応答時間
- **通常**: 5-10分で Pull Request作成
- **複雑な内容**: 最大10分（タイムアウト設定済み）
- **画像込み**: 少し時間がかかる場合あり
